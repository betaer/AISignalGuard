判定口径
评分口径：满分 100，命中即扣；所有已观察到的 IPv4 / IPv6 出口共同判定，出口 IP 取最重一项。

重扣：出口 IP 在中国 -35 · 机房 / VPN 出口 -22 · 大陆直连 -20 · DNS 泄漏中国解析器 -15 · WebRTC 出口外公网候选 -12 · 至少 2 个实际 AI 目标的服务侧国家标签一致命中当前中国口径 -15。

轻扣：信号前后矛盾 -8 · 出口 IP 情报不完整（仅测出地址或完全无法读取）-8 · 语言含中文 -4 · 时区在中国 -4 · Emoji / 字体弱信号各 -1。 多源 IP 情报冲突（至少 3 个地理来源、2 个及以上分歧时）-4。

不适用或证据不足的检测项（如 Windows 上的 Emoji 项、地理来源不足的多源交叉）按中性处理，不扣分也不标绿； 出口 IP 只读到地址但缺少可核对的地区，或完全无法读取，是例外，按上面“情报不完整 -8”计。

Cloudflare 基准只用于观察路径，不参与网络信号参考分；单个 AI 目标命中或同一目标前后标签不一致时按未确认处理，不扣分。 AI 官方服务状态也只用于排查平台故障，不参与网络信号参考分。

本地检测留在浏览器内，联网检测会访问对应第三方服务。

本地项包括语言、时区、Emoji / 字体渲染和浏览器指纹。联网项包括 IP 情报、WebRTC STUN、连通性探测、DNS 泄漏检测、AI 路径和 AI 服务状态。

“复制给AI诊断”生成的结构化报告始终脱敏：IPv4 仅保留前三段、IPv6 仅保留前三组， mDNS、Canvas 与声纹标识不写入原值。右下角“隐藏”只控制页面敏感字段的显示。
